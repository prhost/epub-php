---
Just a package to facilitate Epub generation. I didn't find any good.

## Installation

You can install the package via composer:

```bash
composer require prhost/epub-php
```

## Usage

```php
$epub = new Prhost\Epub\Epub('Title Epub');
```

## Testing

```bash
composer test
```
or
```bash
php vendor/bin/phpunit
```
